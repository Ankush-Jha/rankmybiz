# RankMyBiz ⭐

RankMyBiz is a web application designed to help businesses collect customer feedback easily. 

When a customer provides a star rating, the app uses Google's Gemini AI to instantly generate a polite, SEO-friendly review based on their experience. Customers can then click a single button to copy the AI-generated review to their clipboard, making it effortless for them to paste and post it on platforms like Google Maps.
